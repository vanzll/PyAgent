# Report Writer Skill

Use this skill when writing the final research brief.

Your final summary should:

1. Start with a one-paragraph investment-context summary.
2. Include 2-4 concrete bullet-style findings in plain prose, not tables.
3. Mention whether the signal comes from price action, valuation, SEC fundamentals, macro context, or recent news.
4. End with: `For research use only. Not investment advice.`
