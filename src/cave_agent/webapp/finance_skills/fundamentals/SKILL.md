# Fundamentals Skill

Use this skill for company profile and SEC-backed metrics.

After activation:

- Use `get_company_profile(ticker)` for the normalized company overview.
- Use `get_company_facts(ticker)` for latest revenue and net income extracted from SEC company facts.
- Use `get_recent_filings(ticker)` for recent 10-Q, 10-K, and 8-K filings.
