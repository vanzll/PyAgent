## Product Summary

This project is a public-facing financial research agent built for the NUS CS5260 final project. Users enter one or more U.S. equity or ETF tickers together with a natural-language research question. The system generates a short research brief, snapshot cards, comparison tables, charts, and downloadable artifacts.

## Course Statement

This public-facing agent product is entirely derived from work conducted as part of the NUS CS5260 course.

## Demo Mode And Cost Note

The deployed product supports deterministic demo mode for stable evaluation and low cost. In demo mode, the product does not require paid live market data or paid LLM usage for the default classroom scenarios.
